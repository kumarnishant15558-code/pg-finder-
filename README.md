🏠 **Pg_Finder – Django PG Booking Platform**

Pg_Finder is a full-stack web application built with **Django** that helps users easily find and book Paying Guest (PG) accommodations. The platform provides a seamless experience for both **PG Owners** and **Clients**, enabling efficient listing management and a smooth booking process.

✨ **Key Features**
🔐 Secure authentication with a custom user model and email verification
👤 Role-based access for **PG Owners** and **Clients**
🏠 Owners can create and manage PG listings with pricing, amenities, and images
🔎 Advanced search functionality to find PGs by location or title
📅 Simple and structured **PG booking workflow**
💳 UPI **QR Code payment system** for easy transaction confirmation
📊 Booking lifecycle management (Pending • Confirmed • Cancelled • Completed)
🖼️ Support for multiple images per PG listing

⚙️ **Tech Stack**
🐍 Django (Backend Framework)
🗄️ Django ORM & DBMS
🎨 HTML • CSS (Frontend)
🔑 Custom Authentication System

🚀 Pg_Finder demonstrates real-world **full-stack development concepts** including authentication, role-based systems, relational database design, media handling, and a complete booking workflow.
